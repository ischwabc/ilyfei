Calculation Formulas

Kuz-Ram Model

X50 = A × K-0.8 × Q1/6 × (115/RWS)19/30

R(x) = 100 × (1 - e-(x/X50)n)

Where:

  • X50 = Average Particle Size (cm)
  • A = Rock Factor
  • K = Charge per Unit Volume (kg/m³)
  • Q = Charge per Hole (kg)
  • RWS = Relative Weight Strength of Explosive

Modified Kuz-Ram Model

R(x) = RKR(x) × f(x/xmax)

Where:

  • RKR(x) = Predicted Value from Kuz-Ram Model
  • f(x/xmax) = Correction Factor
  • xmax = Maximum Particle Size

Swebrec Model

P(x) = 100 / (1 + (ln(xmax/x)/ln(xmax/x50))b)

Where:

  • xmax = Maximum Particle Size
  • x50 = Average Particle Size
  • b = Shape Parameter of Curve

Two-Component Model (TCM)

P(x) = ffPf(x) + (1-ff)Pc(x)

Where:

  • ff = Proportion of Fine Particles
  • Pf(x) = Distribution of Fine Particles
  • Pc(x) = Distribution of Coarse Particles
